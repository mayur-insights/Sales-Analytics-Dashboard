# Sales-Analytics-Dashboard

This project focuses on analyzing sales data from the Adventure Works dataset to uncover insights related to revenue, profit, and sales performance. Interactive dashboards were developed to support data-driven decision-making.

Key Insights:
- Analyzed sales trends across years and months
- Identified high-performing periods and sales patterns
- Evaluated profit and production cost relationships

Dashboard Features:
- KPI tracking for total sales, profit, average sales, and cost
- Year-wise and month-wise sales analysis
- Quarterly sales distribution
- Sales vs production cost comparison
- Interactive filters for dynamic analysis

Tools & Technologies:
- Power BI
- Tableau
- Excel
